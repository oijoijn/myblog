Change Key > PowerToys > AutoHotkey > ソフトウェア

2. Change Key

Change Keyを使用することでCtrlとCapsLockを交換できる.欠点としてレジストリをいじることになるので社内のpcなどで使用できない可能性がある.また,1対1のキーに対して行うためショートカットキーの登録のし直しなどができない.

3. PowerToys

PowerToysでショートカットキーの登録のし直しができる.また,Microsoftが提供しているので安心してインストールすることができる.注意点としてPowerToysでCtrlとCapslockを入れ替えると動作が安定しないためChange keyもしくはレジストリを直接書き換えることをおすすめする.

豆知識:Ctrl [でja,eu配列のキーボードどちらもEscキーと似たような動作を行うがEscキーと同じ動作ではないため登録することをおすすめする.

4. AutoHotkey

AutoHotkey はプログラミング言語で上記より自由度の高い設定を行えるがWindows11との相性が悪いのか動作が一番安定しない.しかしマウスの動作制御,キー,ショートカットキーの入れ替えなどができAutoHotkeyでできないことはほとんどないと言える. gif

番外編. Vscode Vim

Vscodeの拡張機能にVimがあります.Vimとはエディターの一種でキーボードだけでカーソルの動画できファイルの編集が可能になります.ターミナルでviやvimと入力することで起動ができる.

番外編. Chrome Vimium

VimiumとはGoogle Chromeに対してVimのような操作を提供する拡張機能である.Shift j, Shift kでタブの移動が可能だがPDFを開いてしまうとタブの移動が行われないそのためPowerToysでShift j,Shift kをCtrl Tab, Ctrl Shift Tabに変更するのをお勧めする.Chromeに対してのショートカットキー
